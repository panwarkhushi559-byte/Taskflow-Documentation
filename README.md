# Taskflow Documentation

This repository contains product documentation for **Taskflow**, a task management platform that helps individuals and teams create, assign, and track tasks.

The documentation is organized to support both end users and developers.

---

## Documentation Overview

### User Guides
User guides help non-technical users get started and manage tasks using the Taskflow web application.

- Quick Start Guide
- Managing Tasks

### API Documentation
API documentation helps developers integrate Taskflow into their applications using a REST API.

- Getting Started
- Authentication
- API Reference
- Errors
- Use Cases

---

## Audience
This documentation is intended for:
- End users who use the Taskflow web dashboard
- Developers who integrate Taskflow using the API
